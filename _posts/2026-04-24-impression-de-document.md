<div style="text-align: justify;"><span>Imprimer une feuille de calcul Excel est un peu délicat. Les données se suivent non seulement de haut en bas, comme dans un texte, mais aussi de gauche à droite. Lorsque le nombre de colonnes est élevé, et qu'il dépasse ce qui peut tenir sur une feuille A3 ou A4, une solution consiste à imprimer en mode Paysage. Mais il faut parfois accoler plusieurs feuilles pour voir l'ensemble des données !</span></div>
<!--more-->
<div style="text-align: justify;"><span>C'est pourquoi l'impression d'une feuille de calcul requiert une attention particulière. Notamment, il est indispensable de prévisualiser l'impression. Autrement, le moindre débordement de colonnes pourrait entraîner un gâchis de papier, car de nombreuses feuilles pourraient ne contenir qu'une seule colonne !</span></div><div style="text-align: justify;"><span>Si en cliquant sur l'onglet Fichier =&gt; Imprimer, le nombre de pages paraît excessif,&nbsp; il faudra sans doute revoir la largeur des colonnes. Il faudra réduire leurs largeurs. Ou utiliser le mode Paysage.&nbsp;</span></div><div><span><br /></span></div><div><table align="center" cellpadding="0" cellspacing="0"><tbody><tr></tr></tbody></table><table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto;"><tbody><tr><td style="text-align: center;"><img border="0" height="246" src="https://blogger.googleusercontent.com/img/a/AVvXsEiH6SVY23G7_oCuzlUHXVvKrB9pBgUg0qKWyoB8mtBLwrgfn7qeKBd2i3_52bhqGJ_g_mPj8utcf7ReuMLgVheubwNePS7xH6fWk7pqOtYdI-pnhIwUcdzMbuhMxHfB6mN5hMdnqc9j_FZLM7A91kqZI0zYS8YMJivzvTeo5go6YIdFcs37AEzuj9q_sRg=w476-h246" style="margin-left: auto; margin-right: auto;" width="476" /></td></tr><tr><td class="tr-caption" style="text-align: center;"><span style="text-align: start;">&nbsp;</span><i style="text-align: start;">Excel affiche l'emplacement des sauts de page par des pointillés</i></td></tr></tbody></table><table align="center" cellpadding="0" cellspacing="0"><tbody><tr><td><span><i>&nbsp;<br /></i></span></td></tr></tbody></table><div><span><i><a href="https://blogger.googleusercontent.com/img/a/AVvXsEiH6SVY23G7_oCuzlUHXVvKrB9pBgUg0qKWyoB8mtBLwrgfn7qeKBd2i3_52bhqGJ_g_mPj8utcf7ReuMLgVheubwNePS7xH6fWk7pqOtYdI-pnhIwUcdzMbuhMxHfB6mN5hMdnqc9j_FZLM7A91kqZI0zYS8YMJivzvTeo5go6YIdFcs37AEzuj9q_sRg"></a></i></span></div><span><i>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;<span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp; &nbsp;</span></i></span></div><div><span>Utiliser le raccourci clavier <i>CTRL + P</i> pour atteindre l'onglet Imprimer. Il faut rappeler que le quadrillage des feuilles n'est jamais imprimé, il suffit de dessiner des bordures autour des cellules avec les commandes de bordure du groupe Police, dans l'onglet Accueil.</span></div><blockquote dir="rtl" style="border: none; margin: 0px 40px 0px 0px; padding: 0px; text-align: right;"><div style="text-align: right;"><div><br /></div></div></blockquote><h3><span style="color: #2b00fe; font-family: arial; font-size: medium;">Imprimer un en-tête ou un pied de page</span></h3><div><ol><li><span>Cliquer sur l'onglet du ruban Mise en page.</span></li><li><span>Dans le groupe du même nom, cliquer sur l'icône Imprimer les titres.</span></li><li><span>Dans la boîte de dialogue Mise en page, cliquer sur l'onglet En-tête/Pied de page.</span></li><li><span>Cliquer sur le bouton En-tête personnalisé. Une boîte de dialogue apparait.&nbsp;</span></li><li><span>Mettre le texte en forme : sélectionner du texte puis cliquer sur ce bouton. Excel affiche la classique boîte de dialogue Police permettant de mettre la typographie en forme.</span></li><li><span>Insérer un numéro de page : Excel insère le code &amp;[Page]. Si vous choisissez le champ &amp; [Page] sur &amp; [Pages], Excel affichera par exemple, en haut à droite de chaque page, 3 sur 7.</span></li><li><span>Insérer la date : Excel insère le code &amp;[Date].</span></li><li><span>Insérer l'heure : Excel insère le code &amp;[Heure].</span></li><li><span>Insérer le chemin et le nom du fichier : Excel insère le code &amp;[Fichier].</span></li><li><span>Insérer le nom de la feuille de calcul : Excel insère le code &amp;[Onglet].</span></li><li><span>Insérer une image : Excel ouvre la boite de dialogue Insérer des images. Rechercher des images dans des emplacements suggérés (à partir d'un fichier présent dans l'ordinateur, depuis un espace OneDrive, depuis un site de cliparts...). Excel insère le code &amp;[Image].</span></li><li><span>Mettre l'image en forme : Ce bouton permet notamment de régler les dimensions de l'image, sa luminosité et son contraste.</span></li><li><span>Remplir un ou plusieurs des trois champs avec du texte et des codes.</span></li><li><span>Cliquer sur OK. Un aperçu est visible dans la zone En-tête.</span></li><li><span>Cliquer sur OK.
